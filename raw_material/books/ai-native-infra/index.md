# AI Native Infrastructure — Systems Designed for Uncertainty

A concise handbook by Jimmy Song targeting CTOs, platform engineers, and technical leaders navigating enterprise AI adoption. The core thesis: AI infrastructure is fundamentally different from cloud-native because models become autonomous actors, GPUs become scarce constrained resources, and uncertainty is the operational default — not an exception to handle.

The book argues for a governance-first (not API-first) approach: build the closed loop connecting intent → execution → resource consumption → cost/risk outcomes before layering on agent capabilities.
